## 1. Storage API

The Web Storage API (including `localStorage` and `sessionStorage`) allows you to store key-value pairs in a web browser. It's useful for saving user preferences or persisting data between sessions.

**Code Example:**

```javascript
// Save data to localStorage
localStorage.setItem('userName', 'Vishal')

// Retrieve data from localStorage
const user = localStorage.getItem('userName')

// Clear localStorage
localStorage.removeItem('userName')
```

[Learn More about Storage API](https://developer.mozilla.org/en-US/docs/Web/API/Storage)

---

## 2. Payment Request API

The Payment Request API simplifies the process of accepting payments on the web by providing a consistent user experience across various payment methods.

**Code Example:**

```javascript
if (window.PaymentRequest) {
  const payment = new PaymentRequest(
    [
      {
        supportedMethods: 'basic-card',
      },
    ],
    {
      total: { label: 'Total', amount: { currency: 'USD', value: '10.00' } },
    }
  )

  payment
    .show()
    .then((result) => {
      // Process payment result
      console.log(result)
    })
    .catch((error) => {
      console.error('Payment failed:', error)
    })
}
```

[Learn More about Payment Request API](https://developer.mozilla.org/en-US/docs/Web/API/Payment_Request_API)

---

## 3. DOM API

The DOM (Document Object Model) API allows you to manipulate the structure, style, and content of the document. This is one of the most widely used APIs in web development.

**Code Example:**

```javascript
// Select and update an element
const element = document.querySelector('#myElement')
element.textContent = 'Hello, World!'
```

[Learn More about DOM API](https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model)

---

## 4. HTML Sanitizer API

The HTML Sanitizer API helps clean up untrusted HTML content to avoid security risks like XSS (Cross-Site Scripting) attacks.

**Code Example:**

```javascript
const dirtyHTML = '<img src="javascript:alert(1)">'
const cleanHTML = sanitizer.sanitize(dirtyHTML)
console.log(cleanHTML) // Safe HTML output
```

[Learn More about HTML Sanitizer API](https://developer.mozilla.org/en-US/docs/Web/API/HTML_Sanitizer_API)

---

## 5. Canvas API

The Canvas API allows you to draw graphics and animations on a web page using the `<canvas>` element, perfect for creating games, visualizations, or custom graphics.

**Code Example:**

```javascript
const canvas = document.getElementById('myCanvas')
const context = canvas.getContext('2d')
context.fillStyle = 'blue'
context.fillRect(10, 10, 150, 100)
```

[Learn More about Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API)

---

## 6. History API

The History API lets you interact with the browser‚Äôs session history, allowing you to manipulate the browser's history stack (e.g., `pushState`, `replaceState`).

**Code Example:**

```javascript
history.pushState({ page: 1 }, 'title', '/page1')
history.replaceState({ page: 2 }, 'title', '/page2')
```

[Learn More about History API](https://developer.mozilla.org/en-US/docs/Web/API/History_API)

---

## 7. Clipboard API

The Clipboard API allows you to read from and write to the clipboard, enabling features like copy-paste functionality.

**Code Example:**

```javascript
navigator.clipboard
  .writeText('Hello, Clipboard!')
  .then(() => {
    console.log('Text copied to clipboard')
  })
  .catch((err) => {
    console.error('Failed to copy text:', err)
  })
```

[Learn More about Clipboard API](https://developer.mozilla.org/en-US/docs/Web/API/Clipboard_API)

---

## 8. Fullscreen API

The Fullscreen API allows you to present a specific element or the entire webpage in fullscreen mode, useful for videos or immersive experiences like games.

**Code Example:**

```javascript
document
  .getElementById('myElement')
  .requestFullscreen()
  .catch((err) => {
    console.error(`Error attempting to enable full-screen mode: ${err.message}`)
  })
```

[Learn More about Fullscreen API](https://developer.mozilla.org/en-US/docs/Web/API/Fullscreen_API)

---

## 9. FormData API

The FormData API simplifies the process of constructing key/value pairs representing form fields and their values for easier form data submission via XHR or Fetch.

**Code Example:**

```javascript
const form = document.querySelector('form')
const formData = new FormData(form)
fetch('/submit', {
  method: 'POST',
  body: formData,
}).then((response) => {
  if (response.ok) {
    console.log('Form submitted successfully!')
  }
})
```

[Learn More about FormData API](https://developer.mozilla.org/en-US/docs/Web/API/FormData_API)

---

## 10. Fetch API

The Fetch API provides a modern and flexible way to make asynchronous network requests, offering a simpler, promise-based alternative to `XMLHttpRequest`.

**Code Example:**

```javascript
fetch('https://api.example.com/data')
  .then((response) => response.json())
  .then((data) => console.log(data))
  .catch((error) => console.error('Error fetching data:', error))
```

[Learn More about Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API)

---

## 11. Drag and Drop API

The Drag and Drop API allows you to implement drag-and-drop functionality in your web applications, enhancing user interactions with intuitive UI elements.

**Code Example:**

```javascript
const item = document.getElementById('item')
item.addEventListener('dragstart', (e) => {
  e.dataTransfer.setData('text/plain', item.id)
})
```

[Learn More about Drag and Drop API](https://developer.mozilla.org/en-US/docs/Web/API/HTML_Drag_and_Drop_API)

---

## 12. Geolocation API

The Geolocation API provides access to geographical location information from the user‚Äôs device, enabling location-based services and features.

**Code Example:**

```javascript
navigator.geolocation.getCurrentPosition(
  (position) => {
    console.log(`Latitude: ${position.coords.latitude}, Longitude: ${position.coords.longitude}`)
  },
  (error) => {
    console.error(`Error getting location: ${error.message}`)
  }
)
```

[Learn More about Geolocation API](https://developer.mozilla.org/en-US/docs/Web/API/Geolocation_API)
